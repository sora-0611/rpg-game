# 画面一覧

| 画面名 | 記載シート名 | 詳細 |
| --- | --- | --- |
| ローディング画面 | セーブ機能関連 | Now loading...を表示。タイトル画面へ移動する。 |
| タイトル画面 | セーブ機能関連 | 設定画面、探索画面へ移動する。 |
| 設定画面 | セーブ機能関連、探索機能関連 | 設定を表示する。設定を変更する。 |
| 探索画面 | セーブ機能関連、戦闘機能関連、強化機能関連、探索機能関連 | キャラクターとHPを表示、プレイヤーを移動、探索、ルート分岐を行う。ハンバーガーメニュー、アイテムウィンドウはこれの手前に表示する。 |
| 戦闘画面 | 戦闘機能関連 | キャラクター、HPを表示。「戦う,防御,アイテム,逃げる」のコマンドを表示する。アイテムウィンドウ、逃げるウィンドウはこの手前に表示する。 |
| 強化画面 | 強化機能関連 | 現在の装備の強化段階を表示する。装備の強化ができる。 |
| マップ選択画面 | 探索機能関連 | 3種類のマップから探索するマップを選ぶ。探索画面に移動する。 |
